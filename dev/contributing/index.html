Contributing

Thank you for your interest in ChenSignatures.jl!

Reporting Issues

We welcome bug reports, feature requests, and suggestions:

  • GitHub Issues: https://github.com/aleCombi/ChenSignatures.jl/issues

When reporting an issue, please include:

  • Julia version (julia --version)
  • ChenSignatures.jl version
  • Minimal reproducible example
  • Error messages or unexpected behavior

Feature Suggestions

If you have ideas for new features or improvements, please open an issue describing:

  • The use case or problem you're trying to solve
  • Your proposed solution or approach
  • Any relevant references or examples from other libraries

Pull Requests

Pull requests are not currently being accepted.

The project is in active development, and the maintainer is focusing on core features and stability. However, your feedback and suggestions via issues are highly valued and will inform future development.

Questions and Discussion

For general questions or discussion:

  • Open a GitHub issue with the "question" label
  • Provide context about what you're trying to accomplish

Code of Conduct

Please be respectful and constructive in all interactions. We aim to maintain a welcoming and inclusive community.
